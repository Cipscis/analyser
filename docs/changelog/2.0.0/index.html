Analyser 2.0.0 - Promises and tidy

Back to changelog

The loadFile method was changed significantly to use the Promise API.

Also, old versions of functions that were previously able to be called on the analyser object itself but had also been made available as methods of the AnalyserRows class have now been moved entirely over to the class.

Breaking changes

  • The loadFile method now returns a Promise and takes an array of fileInfo objects that are expect to include a path string
  • Methods able to be called on AnalyserRows objects can no longer be called on the analyser object itself

Non-breaking changes

  • None
